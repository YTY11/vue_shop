# vue_shop
## 介绍 
* 商品后台管理项目 仿照B站 刘龙彬的 vue教程
* 使用技术 Vue2.x, Element-Ui, Axios, Echarts, DataV
*  [admin 演示地址](http://120.26.52.45:9898/login)
* [码云地址 (国内访问速度快)](https://gitee.com/yty1998/vue_shop.git)
* [GitHub地址](https://github.com/YTY11/vue_shop.git)
